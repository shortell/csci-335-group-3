## Can Elon Musk Tweets help predict Tesla Stock?

## Abstract
Can Elon Musk Tweets Help Predict Tesla Stock? A Sentiment-Based Machine Learning Approach
This study explores whether Elon Musk's Twitter activity can predict short term Tesla stock movements. Using 2,438 filtered tweet price pairs, sentiment scores from a RoBERTa model were combined with pre-tweet stock features to train several classifiers. Logistic Regression, SVM, Random Forest, MLP, and an Ensemble Stacker under both binary and multiclass formulations.
